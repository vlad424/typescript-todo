<h1>TypeScript todo app created with ReactJs<h1/>

<h2>Used:</h2> 
- TypeScript
- ReactJs
- react-redux (^9.0.4) 
- styled-components (^6.1.8)
- react-color (^2.19.3)
- js-cookie (^3.0.5)
- axios (^1.6.7)
- RTK-query

<h2>Features: </h2>
- Create a userboard, stats, nevermind
- Create a user profile interface
- Make a quit account
- Add marks for posts
